Convert Website

Deskshell can be used to take a website, html5 demo or other existing html / javascript application and run it on the desktop. To start create a new blank folder with the name of your application, then create an htdocs folder and copy your website / html5 demo / javascript application into the folder. Then test the application by running it in google chrome.

Deskshell uses an app.js file in json format to describe an application. Here is an example file: 

{
    "name": "myapp1",
    "version": "0.1",
    "author": "email@test.com",
    "description": "My first deskshell app",
    "licence": "MIT",
    "htdocs": "htdocs",
    "frontend": "chromium-portable",
    "backend": "none",
    "width": "300",
    "height": "300",
    "defaultLocation": "index.htm"
}

Place the above file in your application folder with the htdocs folder in it. Click on app.desk to launch htdocs/index.htm inside deskshell. If you application starts with a different file then change the defaultLocation setting in the file.

Thats as hard as it gets if you have a pure html5 / javascript application.

Once you have a working application checkout the application packager and exe compiler to package your application for distribution.
