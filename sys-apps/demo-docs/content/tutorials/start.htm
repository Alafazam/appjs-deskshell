Quick Start

Download the latest deskshell release from deskshell.org

Run Deskshell and click on the "AppWizard" tab and then "Launch App Wizard"


This will open a new window, click on Create New App and fill in a name for the application, your name and a description. Then click Create to make your application.


Clicking on "Launch App" will open your wonder application and "Open App Folder" will open the folder to allow you to view and edit the contents.

Screenshot showing application that was created:


Opening the application folder will show a .desk file, an app.js backend file and then htdocs directory.


The htdocs directory holds your applications front end, if you are mainly going to create an html5 application then simply create your application html, css, js and other content within this directory. If you change the contents then press F5 within the deskshell application to reload the current page and show the updated files. You can use jquery / bootstrap / css3 and whatever else you would like to within this folder.

The app.js file contains javascript that is run by nodejs this provides a "backend" to your application running in deskshell that will allow it to break out of the browser sandbox and access any nodejs functionality (write directly to local disk, access databases, connect via ssh to other machines etc. etc.).

Simply click on the app.desk file to launch your application

To share your application with the world click on the Packager Tab after running Deskshell


Clicking on Package App and then entering the full path to your .desk file will then load information about the application.
Clicking Package App will then package your app for distribution. It will create an .appfs file in the deploy folder. If you now email or ftp this application to others then they can install deskshell and then run your application. For windows users you can create an exe in a similar manner, when run this application will install deskshell if it does not exist and then run your application.

Check out the Demos tab to see some example deskshell applications!
